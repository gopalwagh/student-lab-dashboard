# Student Lab Dashboard

This is a simple lab management project made using HTML, CSS, and JavaScript.

## Features
- Student registration form  
- Dashboard with cards  
- Loading screen  
- Data stored using localStorage  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Live Link
https://gopalwagh.github.io/student-lab-dashboard/

## How to Run
1. Download or clone this repository  
2. Open the folder  
3. Run index.html in your browser
